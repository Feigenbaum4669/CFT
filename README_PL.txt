-----------------------------------"QUANTUM WALKER"-------------------------------------------
Program do klasycznych i kwantowych spacerów na grafach pełnych, cyklicznych i typu lollipop.
Wszystkie spacery rozpoczynają się w wierzchołku 0.
Autor: Oskar Słowik
----------------------------------------------------------------------------------------------

Wymagania:
-Python w wersji >=3.5
-pakiet QISKit (https://github.com/QISKit/qiskit-sdk-py)

Instalacja:
Program nie wymaga instalacji.

Uruchomienie:
Należy wywołać w terminalu program "startWalk.py". Przykładowe wywołanie:
C:\Walker>python startWalk.py
